## About The Project

This project is intended to predict employee turnover.  Because the company has an extended onboarding process due to security 
clearance requirements, predicting when positions will become vacant would allow for proactive recruitment, reducing the length of time between employees
in a position.

### Prerequisites

### Installation

1. Get a free API Key at [https://example.com](https://example.com)
2. Clone the repo
   ```sh
   git clone https://github.com/github_username/repo_name.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Enter your API in `config.js`
   ```js
   const API_KEY = 'ENTER YOUR API';
   ```
